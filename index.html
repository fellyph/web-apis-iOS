01. Geolocation API

The Geolocation API allows the user to provide their location to web applications if they so desire. For privacy reasons, the user is asked for permission to report location information. Example Geolocation
reference

2.Fetch API

The Fetch API provides an interface for fetching resources (including across the network). It will seem familiar to anyone who has used XMLHttpRequest, but the new API provides a more powerful and flexible feature set. Example Fetch API
reference

03. IndexedDB API

IndexedDB is a low-level API for client-side storage of significant amounts of structured data, including files/blobs. This API uses indexes to enable high-performance searches of this data. Example IndexedDB API
reference

04. Stream API

Method of accessing external device data (such as a webcam video stream). Formerly this was envisioned as the device element. Example Stream
